Lost and Found

Finden Sie Ihre verlorenen Gegenstände einfach wieder!

So funktioniert es


Nach einer kostenlosen Registrierung können Sie sich anmelden und verlorenen oder von Ihnen gefundenen Gegenstände veröffentlichen.


Bei diesem Vorgang müssen Sie nur angeben, was für einen Gegenstand, wo und wann Sie gefunden/verloren haben.


Zusätzlich können Sie auch die exakte Veranstaltung, Konzert etc. angeben, damit Ihr Beitrag von anderen Benutzern leichter gefunden werden kann.

Was?

Handy, Geldtasche, Sonnenbrille, Tasche, Kopfhörer, Kleidungsstücke,..

Wo?

An einem Konzert, Festival, auf der Straße - wählen Sie aus der Liste der vorhandenen Veranstaltungen oder fügen Sie eine neue hinzu

Wann?

Geben Sie einfach ein Zeitintervall oder sogar das genaue Datum und den genauen Zeitpunkt an

Sollten Sie Ihr Handy verlieren




Ihr Handy wird gefunden.


Der Finder besucht unsere Website.


Gefundenes Handy wird registriert.


Übereinstimmung zwischen verlorenem und gefundenem Handy wird erkannt.


Beide Seiten werden sofort notifizert, um Rückgabe zu besprechen.
Sie bekommen Ihr Handy zurück!
